电路原理

1.1 电路和电路模型

  • 电路：若干个电器设备和电器原件按一定方式组合起来所构成的电流的通路

  • 电路模型：将实际电路的元器件用理想电路元件适当组合而成的模型

    • 电路四大构件：电源、负载、能量/信号处理电路（传输控制器件）、开关导线
    • 理想电路元件：实际上在不同应用场景同一部件表示不同形式
      • 电阻元件：小号电能的元件
      • 电感元件：产生磁场、储存磁场能量的元件
      • 电容元件：产生电场、储存电场能量的元件
      • 电容元件：将其它形式的能量转变为电能的元件

  • Regard circuits:

    • Load

      • Resistive
      • Dynamic
        • capacitance
          • transient
          • steady state

    • Source

      • DC
      • AC
        • Sine
        • Periodical

  • 电路的分类：

    1. 线性：激励与相应之间为线性关系

      • 满足齐次性和可加性

    2. 非线性：只要有一个或多个负载不满足线性关系

    3. 平面电路：总有一次重画出现元件之间不会有交叠关系

    4. 非平面电路：不论如何重画，总会有交叠关系的存在

  • 支路变量：

    • 电流：带正电粒子有规则的定向运动

      • AC：随时间变化
      • DC：不随时间变化

    • 电压：电场力在单位正电荷移动时所做的功

      • 单位正电荷所吸收的能量

    • 电位：某一点到参考点的电压

      • 参考点的电位为ZERO

    • 电动势：非电场力将单位正电荷移动时所作的功

      • 电压区别：电动势使某电位具有能量提升

    • 支路变量参考方向:主要是电桥，正弦激励

      • 电流

        • 表示方法

          1. 箭头
          2. 下标

        • 实际方向

          • i > 0 ： 参考方向与实际方向相同

          • i < 0 ： 参考方向与实际方向不同
            $$
            i = \frac{dq}{dt}
            $$

      • 电压

        • 表示方法

          1. 箭头
          2. 正负号
          3. 双下标

        • 实际方向

          • u > 0 ： ~

          • u < 0 ： ~
            $$
            u = \frac{dw}{dq}
            $$

      • 电动势

        • 表示方法：
          1. 箭头
          2. 正负号
          3. 双下标

    • 关联参考方向：

      • 关联：电流从电压正端流入
      • 非关联：电流从电压负端流入

    • 功率：单位时间内能量的变化

      • $$
        P = \frac{dw}{dt}=\frac{dw}{dq}·\frac{dq}{dt}
        $$
- $$
  P_d = P_a
  $$

- 关联：absorb

  - P > 0 ：吸收
  - P < 0 ：放出

- 非关联：deliver
  - P > 0 ：放出
  - P < 0 ：吸收
