    • 注释

      会敲代码的人真的不一定看得懂自己的代码，同样的，看得懂代码的人也不一定会敲代码。

      如高中时期，有时候看数学的大题答案时，还真不一定知道某一部分的公式为什么要这样转换，如果答案下面附带着解析，那就更方便于理解答案了。

      所以，注释的作用就是解析的作用，也是程序员用来以后自己查看代码时能够迅速回想起当时编写时的逻辑，也能帮助他人理解自己的代码。

    函数语句块：函数定义或者类定义

    • 函数定义

      函数是用于包含逻辑程序代码并拥有其自身功能的结构

      如数学中定义函数 f(x) = x+1

      其中x+1在程序中为逻辑代码，即函数f(x)所包含的”程序“代码

      函数在代码中的地位：

      正常的逻辑程序代码都是需要写在其相对应的函数中的，程序的执行，总是需要一个起点作为入口的，在C/C++中，程序的入口即main()函数。而python中，并没有定义main()数作为入口，而是直接写逻辑程序即可逐行执行。但实际上，如果一段逻辑程序代码不包含于任何函数中，是可以理解为这些程序代码是包含于main()函数中的，程序的入口，仍然还是main()函数。

  • 标准数据类型

    Python中包含6个标准的数据类型：

    • Number (数字)
    • String (字符串)
    • List (列表)
    • Tuple （元组)
    • Set (集合)
    • Dictionary (字典)

    六个标准数据类型又可以下划分：

    • 不可变数据：Number 、String 、Tuple

    • ​ 可变数据：List、Dictionary、Set

      深入数据类型的推荐学习《数据结构与算法》

  • 基本数据类型

    • Number

      Python3 支持四种Number类型：int 、float 、bool 、complex

      • int ：整形，初学可理解为整数类型

        支持十进制、二进制(OB)、八进制(OO)、十六进制(Ox)

      • float ：浮点型，初学可理解为浮点类型

      • bool ：布尔型，初学可理解为条件类型(真或假、是或否)

        bool取值范围是：False 、True 两个关键字

        在C语言和python2中不存在这种类型，则用数字1表示True，用数字0表示False

      • complex ：复数型，顾名思义
